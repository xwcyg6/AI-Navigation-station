🦉 AI新闻

🚀 ChatGPT只会25个老掉牙笑话，不会自创幽默

摘要：最新研究发现，ChatGPT聊天机器人虽然可以回答各种问题，但不懂幽默，只会重复25个老掉牙笑话，且无法创造新笑话。两位德国研究者在arXiv网站发表了一篇论文，通过让ChatGPT生成1008个笑话，发现其中90%以上都是同样的25个笑话，无法理解其中的意思且不能自主创新。ChatGPT在构建笑话的结构方面显示出了一些进步，但还不能自信地创造有意思的原创内容。

🚀 四个00后开源计划：将整个互联网转成Tokens，1亿Token嵌入成本只需要1美元

摘要：四个平均年龄20岁的少年创建了名为“宏观世界联盟”的开源计划，旨在将整个互联网转化为Tokens数据集。他们已经将Arxiv上的所有论文转换成Token，并推出了嵌入搜索平台，嵌入数据集的成本极低。目前每1亿个Token嵌入成本仅需要1美元。该数据集可为GPT-4等大型模型打造插件提供助力，帮助学习、教学和科研。

🚀 iPhone 上也能跑大型语言模型了，陈天奇团队推出 MLC LLM

摘要：MLC LLM是陈天奇团队推出的用于在移动设备上运行大型语言模型的开源项目，支持iOS和安卓手机，并可以在Windows和Mac台式机上运行。目前已开放测试，用户可以测试已有的模型或自己构建模型。该项目开启了大型模型私有化部署的可能，拓宽了语言模型在移动设备上的应用。

🚀 OpenAI 竞争对手 Cohere 宣布获得 2.7 亿美元融资，准备与 OpenAI 展开正面竞争

摘要：创立于2019年的Cohere宣布获得2.7亿美元的新融资，该公司于2020年推出了基于云计算的自然语言处理AI平台，适用于企业级应用。该公司CEO称，“当下AI正在落实商业阶段，而下一阶段的AI产品和服务将彻底改变商业的未来，我们已准备好引领潮流” 。此次融资由Inovia领投，英伟达、甲骨文、Salesforce Ventures 等其他公司跟投，Cohere公司估值已达到22亿美元。

🚀 新华三发布私域大模型“百业灵犀”LinSeer，打通垂直应用数据为客户提供智能化服务

摘要：新华三发布了私域大模型“百业灵犀”LinSeer，该模型为垂直行业和专属地域的客户提供安全、订制、独享、生长的智能化服务。该大模型打通垂直应用数据，形成精准、精确、精益的私域垂直智能，训练时间比以往缩短了70%。新华三还发布专为大模型训练而生的AI服务器、全球首发51.2T、800G硅光数据中心交换机、支持“百业灵犀 + 新华三ICT基础构架”与“客户自选大模型 + 新华三ICT基础构架”两种模式。除此之外，新华三还发布了多款新品，覆盖算力资源供给、网络联接质量、绿色节能要求、数据处理效率、安全防护能力、统一运维体验六大能力。
